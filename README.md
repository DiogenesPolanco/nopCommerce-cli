## Development Setup

### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]

### Setting Up a Plugin of nopCommerce

Install the nopCommerce CLI globally:

```
npm install -g nopcli
```

Create workspace:

```
nopcli new [PLUGIN NAME]
```

Run the application:

```
cd [PLUGIN NAME]
nopcli build
```
